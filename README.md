# Komado Scale Adjustor

こまどアバター（ショコラ、シフォン、プラム、ライム）間で、衣装のスケールを自動調整するUnityエディタ拡張機能です。
共通素体を持つアバター間の着せ替え作業を、ヒエラルキー上の右クリックメニューからワンタッチでサポートします。

## 対応アバター
以下の共通素体アバター間のスケール変換に対応しています。
* ショコラ (Chocolat)
* シフォン (Chiffon)
* プラム (Plum)
* ライム (Lime)

## 導入方法

1. GitHubの [Releases](../../releases) ページから、最新の `KomadoScaleAdjustor.unitypackage` をダウンロードします。
2. Unityプロジェクトを開いた状態で、ダウンロードした `.unitypackage` をダブルクリックするか、UnityエディタのProjectウィンドウにドラッグ＆ドロップします。
3. インポートウィンドウが表示されるので、すべてチェックが入っていることを確認して `Import` をクリックします。

※本ツールは `KomadoScaleAdjustor.cs` と `KomadoScaleSettings.cs` の2つのスクリプトで動作します。

## 使い方

1. Unityの `Hierarchy` ウィンドウで、スケールを変換したい衣装などのオブジェクトを選択します（複数選択可）。
2. 右クリックし、メニューから `KomadoScaleAdjustor` を選択します。
3. `From 〇〇`（元の衣装の対応アバター）を選び、次に `To 〇〇`（着せたいアバター）を選択します。
4. 対象オブジェクトの XYZ スケールに、指定したアバター間の適切な倍率が自動で乗算されます。

### 追加のスケール設定
アバター間の基本倍率に加えて、さらに任意の倍率を掛け合わせたい場合は設定ファイルを使用できます。

1. 右クリックメニューから `KomadoScaleAdjustor` -> `⚙️ Settings` を選択します。
2. インスペクターに設定ファイルが表示されます。
3. `Target Avatar Scale` の数値を変更すると、次回のスケール変換時からその倍率が加味されます。

## 注意事項
* 本ツールはスケールの調整のみを行う補助ツールです。
* 衣装の形状によっては、スケール調整後に手動での位置合わせやウェイト調整、メッシュの微調整が必要になる場合があります。
